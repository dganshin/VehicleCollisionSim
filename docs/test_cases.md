## 当前重点关注问题

- 车辆在当前参数下是否已经具备足够明显的前进与转向效果。
- 相机跟随是否已正确挂载并指向 `bigyellowbee`。
- 碰撞时是否存在穿模、抖动或过度弹飞。
- 后续 UI 面板接入后，参数与物理表现是否一致。
